# dashboard_obras_gc
Análises dos gastos e o andamento das obras em Graccho Cardoso-SE
<div>
  <h1>Análise das Obras Públicas de Graccho Cardoso em 2025</h1>
  <img src="https://github.com/SilvaMarcosJoao/dashboard_obras_gc/blob/main/ImagemProjeto.png"/>
  <h2>Objetivo</h2>
  <p>O intuito do projeto é ter um panorama geral dos gastos públicos em obras públicas no cidade, distribuição geográfica das obras, andamento e tempo gasto para realizá-las.</p>
  <h2>KPI's Desenvolvidos</h2>
  <ul>
    <li>Valor Total dos Contratos Firmados</li>
    <li>Valor Total das Obras</li>
    <li>Total de Obras</li>
    <li>Duração Média das Obras</li>
    <li>Obras por Classificação</li>
    <li>Situação das Obras</li>
    <li>Obras por Região</li>
    <li>Valor de cada obra por local</li>
  </ul>
  <h2>Insights Extraídos</h2>
  <ul>
    <li>Mais de 70% das obras iniciadas já foram concluídas</li>
    <li>85% dos investimentos foram em escolas e unidades de saúde</li>
    <li>Os Povoados foram as regiões mais beneficiadas</li>
  </ul>
  <h2>Ferramentas Utilizadas</h2>
  <ul>
    <li>Power BI - Limpeza,Modelagem e apresentação dos dados</li>
    <li>Excel - Fonte de dados</li>
  </ul>
